## üìù Description
A minimal, beginner-friendly personal expense tracker that helps users record, track, and analyze income and expenses. Users can register and log in, add transactions (income or expense), filter transactions by date or category, edit or delete entries, and view a categorical summary visualized using a pie chart. The app uses JWT for authentication and MongoDB for persistence.

**Key user flows**
- Register ‚Üí Login ‚Üí Add transaction ‚Üí View list & summary
- Filter transactions by date range or category
- Edit / Delete transactions
- Visual summary (category-wise) via a pie chart

## Overview
A minimal full-stack web app that helps users record and track their daily expenses and incomes.  
Built with:
- **Frontend:** React + Vite + Bootstrap  
- **Backend:** Node.js + Express + MongoDB (Mongoose)  
- **Auth:** JSON Web Tokens (JWT)
