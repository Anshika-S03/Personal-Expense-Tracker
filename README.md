#  Personal Expense Tracker

**Author:** Anshika Sharma  

Live demo:
- **Frontend:** https://personal-expense-tracker.vercel.app  
- **Backend API:** https://personal-expense-tracker-2ve0.onrender.com  

---

## Overview
A minimal full-stack web app that helps users record and track their daily expenses and incomes.  
Built with:
- **Frontend:** React + Vite + Bootstrap  
- **Backend:** Node.js + Express + MongoDB (Mongoose)  
- **Auth:** JSON Web Tokens (JWT)

---

## Features
✅ User registration & login  
✅ Add, edit, and delete transactions  
✅ Filter by date or category  
✅ View total summary and pie chart visualization  
✅ Secure authentication using JWT
